Limitations and future work:

One of the limitations right now is that the ABAQUS.rpt only contains the deflection values. Now, we intend to create a fitness function that will maximize the bending and minimize the prescribed pressure. Although we did create a history output that contains the pressure values as well as the deflection values, the python script was unable to create the report file via history output. It only allowed us to create .rpt files using field output variables which does not have the pressure change. So, for now, our pressure is constant and the shape of the finger changes based on the maximum deflection only. In near future, we will try to evolve our fitness functions (bending angle and pressure) and update the NSGA code subsequently. We will even add a blocking force fitness function and will try to maximize that as well. Hopefully, after this, we will try to optimize actuators with complex geometries for future research work.
